# Password Identify

### 功能描述

​	这个项目的主要功能是密码鉴定，用于注册页面，鉴定用户输入的密码安全性以及是否符合规范。

### 开发环境

​	**操作系统**：Windows 10

​	**IDE**: eclipse

### 测试步骤

​	测试 Demo 放于 `src/test` 下，使用 `jenkins + Ant + Junit` 进行持续集成测试。

​	`jenkins` 地址： `localhost:8080/`
